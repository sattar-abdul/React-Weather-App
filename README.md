## React based Weather App
Using Material UI

### Development
1. Install dependencies
2. Create `.env` with `OPENWEATHER_API_KEY`
3. Run `npm run dev` or `vercel dev`

### Deploy to Vercel
1. Push to GitHub
2. Import the repo in Vercel
3. Set env var `OPENWEATHER_API_KEY`
4. Deploy

### API Routes (Vercel)
- `GET /api/weather?city=CityName`
- `GET /api/forecast?city=CityName`
